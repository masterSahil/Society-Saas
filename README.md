## 🚀 What is this?
Welcome to **SocietySync**! If you've ever lived in an apartment complex or housing society, you know how chaotic things can get—managing visitor entries, collecting maintenance bills, and keeping everyone updated. 

SocietySync solves all of that. It is a powerful yet beautifully simple web application designed to bring peace of mind to residents and staff alike. With just a few clicks, you can manage everything securely and efficiently!

## ✨ Key Features
- **Smart Visitor Management:** Track who enters and leaves with intelligent access controls.
- **Automated Billing:** No more manual math. Generates maintenance invoices automatically.
- **Helpdesk & Ticketing:** Residents can raise complaints easily, and admins can resolve them quickly.
- **Community Announcements:** Keep everyone in the loop with digital notice boards.
- **Premium User Interface:** Designed with stunning glassmorphism, vibrant colors, and smooth animations to provide an amazing experience.

## 🏃‍♂️ How to Run This Project on Your Computer

*Even if you have zero coding knowledge, just follow these simple steps!*

1. **Prerequisites:** Make sure you have [Node.js](https://nodejs.org/) installed on your computer.
2. **Open Terminal:** Open your command prompt or terminal in this project's folder.
3. **Install Dependencies:** Type the following command and hit Enter:
   ```bash
   npm install
   ```
4. **Start the App:** Once installation is done, type this and hit Enter:
   ```bash
   npm run dev
   ```
5. **View the Magic:** Open your web browser and go to `http://localhost:3000`. You will see the beautiful login page!
